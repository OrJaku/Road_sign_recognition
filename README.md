# Road sign recognition [ Praca magisterska ]


Praca związana z rozpoznawaniem elementów infrastruktury drogowej (znaków) przy użyciu technik uczenia maszynowego - deep learningu. 

Gówne technologie:
- Python,
- Keras, 
- OpenCV,
- Google Colab - Jupiter Notebook.
